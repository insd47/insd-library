# React UI Library by insd.

개인 프로젝트 등에 사용하는 React UI 라이브러리입니다.

## 개발 방법

이 프로젝트는 Storybook을 사용합니다.\

1. 해당 프로젝트를 클론합니다.

```
git clone https://github.com/insd47/insd-library
```

2. 외부 npm 모듈을 설치합니다.

```
yarn install
```

3. 프로젝트를 실행합니다.

```
yarn storybook
```

## 제작 완료 컴포넌트

- Button
- Disabler
- Icon
- Loading
- Pressable
- Textfield
- Tab
- Boolean (Radio + Checkbox)
- Tooltip (Story 없음)
