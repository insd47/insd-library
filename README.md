# React UI Library by insd.

개인 프로젝트 등에 사용하는 React UI 라이브러리입니다.\
자체적인 인터페이스 구성 규칙이 있기 때문에, 외부 베포는 매우 어렵습니다.

## 사용법

이 프로젝트는 Storybook을 사용합니다.\

1. 해당 프로젝트를 클론합니다.

```
git clone https://github.com/insd47/insd-library
```

2. 외부 npm 모듈을 설치합니다.

```
yarn install
```

3. 프로젝트를 실행합니다.

```
yarn storybook
```
